Flatten a nested list structure.

Define a function flatten which returns a new "flat" list by replacing each "sub-list" with its elements. 

>>> a_list = [0, 1, [2, 3], 4, 5, [6, 7]]
>>> flatten(a_list)
[0, 1, 2, 3, 4, 5, 6, 7]
>>> another_list = [['a', ['b', ['c']]], 'd', [['e']], 'f']
>>> flatten(another_list)
['a', 'b', 'c', 'd', 'e', 'f']
