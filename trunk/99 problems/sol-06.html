Find out whether a list is a palindrome: solution

A simple solution to see if a list is a palindrome is to compare it with a second identical list - but reversed. Testing equality (==) of two lists is equivalent to verifying that both lists have the same elements in the same location; however, it is much easier than doing the same verification explicitely, one element after the other.

Run the following doctest to verify that the solution given works as expected.

>>> def is_palindrome(some_list):
...     copy = some_list[:]
...     copy.reverse()
...     return some_list == copy
>>>
>>> a_list = ['s', 'p', 'a', 'm']
>>> is_palindrome(a_list)
False
>>> another_list = ['k', 'a', 'y', 'a', 'k']
>>> is_palindrome(another_list)
True
