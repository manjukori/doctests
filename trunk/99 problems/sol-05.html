Reverse a list: solution

Reversing a list is easy in Python using the reverse() list method. The only tricky thing to remember is that it reverses a list "in place" instead of returning a new list.

Run the following doctest to verify that the solution given works as expected.

>>> def reverse(some_list):
...     some_list.reverse()  # reverse a list "in place"
...     return some_list
>>>
>>> a_list = [0, 1, 2, 3, 4, 5, 6, 7]
>>> reverse(a_list)
[7, 6, 5, 4, 3, 2, 1, 0]
>>> another_list = ['a', 'b', 'c', 'd', 'e', 'f']
>>> reverse(another_list)
['f', 'e', 'd', 'c', 'b', 'a']
