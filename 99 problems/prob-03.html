Find the Kth element of a list

Define a function element_at which returns the Kth element of a list; the convention used for this problem is that K=1 corresponds to the first element.

>>> a_list = [0, 1, 2, 3, 4, 5, 6, 7]
>>> element_at(a_list, 3)
2
>>> another_list = ['a', 'b', 'c', 'd', 'e', 'f']
>>> element_at(another_list, 1)
'a'
